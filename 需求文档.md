# 50languages.com 爬虫系统设计文档

## 1. 项目概述

爬取 [50languages.com](https://www.50languages.com/) 的所有语言对及学习内容（纯文本）。

### 数据规模估算
- **语言数量**: 57种语言（含英语UK/US两个变体）
- **语言对数量**: 57 × 56 = 3,192 对
- **每对课程数**: 100 个主题课程
- **每课程短语数**: ~18 个短语
- **总短语估算**: 3,192 × 100 × 18 ≈ **5,745,600 条短语**

## 2. 网站结构分析

### 2.1 URL 模式

| 类型 | URL模式 | 示例 |
|------|---------|------|
| 语言对列表 | `/{ui_lang}/learn/phrasebook/{target_lang}` | `/en/learn/phrasebook/de` |
| 课程详情 | `/{ui_lang}/learn/phrasebook-lessons/{lesson_id}/{target_lang}` | `/en/learn/phrasebook-lessons/162/de` |

### 2.2 课程ID映射
- Lesson 001 = ID 162, Lesson 002 = ID 163, ..., Lesson 100 = ID 261
- **公式**: `lesson_id = lesson_number + 161`

### 2.3 技术约束
- 课程详情页需要JavaScript执行（直接HTTP请求返回403）
- 存在 Cloudflare 防护，需使用 scrapling 过 CF
- 无公开API端点

### Kafka Topic 设计

| Topic | Partition Key | 说明 |
|-------|--------------|------|
| `crawl_tasks` | `{ui_lang}_{target_lang}` | 课程详情爬取任务 |

**消息格式**:
```json
{
    "ui_lang": "en",
    "target_lang": "de",
    "lesson_id": 162,
    "lesson_number": 1,
    "retry_count": 0
}
```

## 4. 数据库设计

### 唯一数据表

```sql
CREATE TABLE phrases (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    ui_lang VARCHAR(10) NOT NULL COMMENT '界面语言代码',
    target_lang VARCHAR(10) NOT NULL COMMENT '目标语言代码',
    native_text TEXT NOT NULL COMMENT '原语言句子',
    target_text TEXT NOT NULL COMMENT '目标语言句子',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_lang_pair (ui_lang, target_lang)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

## 5. 生产者设计

### 职责
1. 爬取首页提取57种语言列表
2. 生成3,192个语言对
3. 对每个语言对爬取课程列表页，获取100个课程ID和主题
4. 发送消息到 Kafka `crawl_tasks` topic

### 爬取流程
```
Step 1: GET / → 解析语言列表
Step 2: 生成语言对笛卡尔积
Step 3: 对每对 GET /{ui}/learn/phrasebook/{target} → 解析课程列表
Step 4: 发送消息到 Kafka (ui_lang, target_lang, lesson_id)
```

---

## 6. 消费者设计

### 职责
- 多 Worker 从 Kafka 消费任务
- 用 scrapling 过 CF 后提取短语
- 存入 MySQL `phrases` 表

### Worker 工作流程
```
1. 从 Kafka topic crawl_tasks 消费消息
2. scrapling 请求课程详情页（自动过 CF）
3. 解析页面提取短语对 (native_text, target_text)
4. 批量 INSERT INTO phrases
5. 提交 Kafka offset
```

### scrapling 配置
- 使用 scrapling 的 Fetcher（支持 JS 渲染 + CF bypass）
- 自动重试 CF challenge
- 随机 User-Agent
